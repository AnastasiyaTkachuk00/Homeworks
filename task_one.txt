1)Title: Сделать Fork репозитория
2)Type: Functional
3)Priority: Medium
4)Estimate: 10m
5)Automation type: none
6)Preconditions: У пользователя есть учетная запись в GitHub 
7)Steps: 1.Войти в свою учетную запись на сайте https://github.com/
         2.Cкопировать url-ссылку репоризитория для которого вы хотите создать копию
         3.Вставить url-ссылку репозитория в вашем браузере
         4.Нажать кнопку Enter
         5.В правом верхнем углу нажать кнопку Fork
         6.Нажать кнопку Create fork
8)Expected result:4.Откроется страница репозитория
                  6.Кол-во значений около кнопки fork изменилось
                    Также в названии репозитория отобразилось ваше имя пользователя
                    Под именем репозитория отображается текст 'forked from{имя скопированного репозитория}'
