1)Title: Закрытие Pull Request ветки «test» в ветку «master»
2)Type: Functional
3)Priority: Medium
4)Estimate: 15m
5)Automation type: none
6)Preconditions: У пользователя есть учетная запись в GitHub 
7)Steps: 1.Войти в свою учетную запись на сайте https://github.com/
         2.Перейти на вкладку Pull Requests, которая отображается в шапке страницы
         3.Выбрать нужный pull request и перейти к нему нажав на его название
         4.Проскроллить вниз страницы до зеленой кнопки Merge pull request
         5.Нажать кнопку Merge pull request
         6.Нажать кнопку confirm merge
8)Expected result:
         3.Пользователю отображается список созданных pull requests
         6.После нажатия на кнопку отображается блок с текстом: 
         Pull request successfully merged and closed
